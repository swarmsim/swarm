Patch Notes

See the source code repository for a more thorough, but less readable, change history.

0.1.25 2014/09/14

  • Adjusted a late-game upgrade's visibility.
  • Minor export format change.
  • Minor sidebar formatting changes.

0.1.24 2014/09/14

  • Removed 0.1.22's grid UI.
  • Wide screens are wider.
  • Another UI overhaul: tabbed sidebar. We're getting closer to something I'm happy with; the UI should be more stable soon - thanks for being patient.
  • If you love scrolling, enable "advanced unit data" in options and look for the all-units tab in the menu.

0.1.23 2014/09/13

  • Selecting a unit now continues to show that tab's unit table, instead of next/previous navigation.
  • Unit selection no longer tries to fill the entire screen. (Oops.)

0.1.22 2014/09/12

  • Added an experimental, unpolished new(er) interface. Let me know what you think of it - it might replace tabs.

0.1.21 2014/09/12

  • Removed swipe navigation - no more accidentally moving to a different unit because you moved the mouse wrong.
  • You'll now see an indicator when you can afford a new upgrade. It should be hidden for unwanted upgrades - looking at a unit/buy-dropdown without buying an available upgrade will make the indicator for that type of upgrade disappear.
  • Reformatted tables a bit.

0.1.20 2014/09/11

  • New UI! Units are now grouped into tabs instead of one long sidebar, among other changes.

0.1.19 2014/09/10

  • Fixed a bug with larva visibility.
  • Altered some early units' visibility to better match the tutorial text.

0.1.18 2014/09/08

  • Some achievements are unmasked as you get closer to completing them.
  • Fixed achievement link color.
  • Masked achievements ("???") can now be shown/hidden.

0.1.17 2014/09/07

  • Achievements!
  • Added some missing unit descriptions.

0.1.16 2014/09/05

  • Changelog link moved next to the other links. Maybe it'll be clearer that it's a link now, and people will actually see this page.
  • Fixed cocoon typo.

0.1.15 2014/09/05

  • Added a new larva upgrade, Cocooning, required to buy a new unit, the Cocoon. If you're trying to save up lots of larvae, encase them in cocoons for safekeeping so you don't accidentally spend them. Cocoons should let you use the "buy max" button without fear.

0.1.14 2014/09/04

  • The game's non-https address should now load properly in Firefox.
  • The options screen lets you opt out of Google Analytics.

0.1.13 2014/09/04

  • You can now specify a much higher number of units to buy at once - it doesn't max out at 1 sextillion anymore.

0.1.12 2014/09/04

  • Invisible backend changes, nothing to see here.

0.1.11 2014/09/03

  • Upgrade costs should have more consistent order.
  • Nicer, more consistent number formatting.
  • The page refreshes automatically when a newer version of the game is detected.

0.1.10 2014/09/02

  • Inject Larvae works again. Oops.

0.1.9 2014/09/01

  • When buying multiple units, their total cost is shown instead of the cost for a single unit.
  • The cost of the maximum number of units you can afford can now be shown as a percentage. This is off by default, visit the options menu to enable it.

0.1.8 2014/09/01

  • Buy-exactly-n is back by popular demand.
  • Units-bought counts include twins again. (Oops.)
  • Top secret voodoo magic.

0.1.7 2014/08/31

  • You can now buy multiple upgrades with one click.
  • There are now three buttons for buying units: buy 1, buy 25%, buy max. Buy-exactly-n is gone; it was awkward to use anyway.
  • Removed the buy/select mode from the top. It wasn't widely used, and was confusing if accidentally changed.

0.1.6 2014/08/31

  • More invisible backend changes, nothing to see here.

0.1.5 2014/08/31

  • Selecting a unit now shows how fast it's being produced. Statistics page has this too.
